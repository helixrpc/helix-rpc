# Helix RPC Rust SDK

Helix RPC is a high-performance AI Gateway and Dynamic Batching framework.
This crate provides the `helix-rt` Rust SDK, allowing you to build pure Rust AI Gateways that feature:
- Native Server-Sent Events (SSE) streaming.
- Production middlewares like `grpc-timeout` and `gzip`.
- `PyO3` integration for Zero-Serialization Python bindings (embedding Python models directly in the Rust memory space).

Please visit the [main repository](https://github.com/helixrpc/helix-rpc) for comprehensive documentation and usage examples.
